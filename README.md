#RishiSphere

(One Calendar with every Event)

RishiSphere is a modern campus event management platform designed to centralize all university events into one seamless ecosystem.

Official events, club activities, workshops, hackathons, cultural fests, admin announcements — everything lives in one intelligent platform.

No WhatsApp chaos.  
No missed opportunities.  
No event clashes.  
No student left behind.

#Features

#Student Experience
- Discover upcoming campus events
- One-click event enrollment
- Personalized dashboard
- Smart event browsing
- Centralized campus calendar
- Real-time event visibility

#Club & Organizer Features
- Create and manage events
- Publish official/unofficial events
- Track registrations
- Attendance management
- Conflict-aware scheduling

#Admin Features
- Monitor complete campus activity
- Prevent scheduling clashes
- Campus-wide analytics
- Event moderation
- Unified event ecosystem

#Tech Stack
# Frontend
- React.js
- Vite
- Tailwind CSS
